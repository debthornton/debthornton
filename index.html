Fansí Schmansí

Fansí Schmansí is a demonstration of CSS, SASS, Object Oriented and Functional JS Programming on a fictional restaurant website.

This was an exercise in responsive font-end web development, UI design, and Google Maps API integration. Vanilla JS was used in order to prevent the reliance of libraries and frameworks.

HTML5 / CSS3 / SASS / JS

Syzygy

Syzygy is a web app that joins productivity and relaxation into one simple UI. The app displays a 12-hour clock on non-mobile displays, an editable todo-list, and an audio control interface that interacts with the Soundcloud API.

Background photos change according to the currently selected sound. Creating this app required extensive functional programming. I built the functionality of Syzygy using Vanilla JS in order to better acquaint myself with the ins-and-outs of the language in its purest form.

I use Syzygy everyday, and it has become my favorite project as a result.

HTML5 / CSS3 / JS

Film Scope

Film Scope is a web app where users can search for film details by title. The app accesses the OMDB API and returns matching responses as an array of movie objects. Details such as the plot, writer/director, and cast are returned as well in an easy to read UI.

Creating Film Scope was an exercise in API accessibility.

HTML5 / CSS3 / BOOTSTRAP / JS / JQUERY

Roman Numeral Convertor

Roman Numeral Convertor is a simple web app that does what the name says. It was developed using Vanilla JavaScript, CSS, and SASS.

HTML5 / CSS3 / SASS / JS

About Me

Code has always been something that I've dabbled in since childhood, whether it was creating small webpages for class or cooking up mods for the Sims! In 2017, during my junior year of college, I decided to delve into coding more deeply. I haven't been able to separate myself since.

That same year, I began creating webpages for classmates, friends, and family both recreationally and professionally. Seeing the result of a program, snippet, app, or even a small piece of code in action is one of the most rewarding experiences I've had.

After college I continued to study code independently, focusing on web development. Each project hosted on this website was completed as a challenge to myself and in order to further develop either a programming concept, a framework, or my ability to create with the tools at hand.

It's been a blast so far. My time at my desk with an editor open is genuinely my favorite part of the day, challenges and all. I also enjoy mixed martial arts, playing my guitar, and singing to my dog, Cammy.
